2. Update next.config.mjs:
   - Import `withPlausibleProxy` from 'next-plausible'
   - Import `withBundleAnalyzer` from '@next/bundle-analyzer' (dynamic import for ESM compatibility)
   - Create bundleAnalyzer config with:
     - enabled: `process.env.ANALYZE === 'true'`
     - openAnalyzer: false (generate files only, don't auto-open browser)
   - Chain configs: withBundleAnalyzer(withPlausibleProxy()(withMDX(nextConfig)))
   - Keep existing MDX configuration intact

Example structure:
```javascript
import createMDX from '@next/mdx';
import { withPlausibleProxy } from 'next-plausible';

const withBundleAnalyzer = (await import('@next/bundle-analyzer')).default({
  enabled: process.env.ANALYZE === 'true',
  openAnalyzer: false,
});

const nextConfig = {
  pageExtensions: ['js', 'jsx', 'md', 'mdx', 'ts', 'tsx'],
};

const withMDX = createMDX({});

export default withBundleAnalyzer(withPlausibleProxy()(withMDX(nextConfig)));
```

3. Add npm script to package.json:
   - "analyze": "ANALYZE=true npm run build"
