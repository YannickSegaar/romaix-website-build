1. Install shadcn/ui Accordion component:
   ```bash
   npx shadcn@latest add accordion
   ```
   This creates `src/components/ui/accordion.tsx` with AccordionItem, AccordionTrigger, AccordionContent exports.

2. Add CSS marquee animation to `src/app/globals.css`. Add AFTER the existing @import statements and @theme block, BEFORE any other styles:
   ```css
   @keyframes marquee {
     0% {
       transform: translateX(0);
     }
     100% {
       transform: translateX(-50%);
     }
   }

   .animate-marquee {
     animation: marquee 30s linear infinite;
   }

   .animate-marquee:hover {
     animation-play-state: paused;
   }
   ```

Note: Use CSS-only animation (not Framer Motion) for logo marquee - better performance, runs on compositor thread.
  </action>

2. Verify accordion file exists: `ls src/components/ui/accordion.tsx`
