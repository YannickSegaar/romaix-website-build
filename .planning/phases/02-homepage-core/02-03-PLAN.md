```typescript
              <CardHeader>

                <div className="text-4xl md:text-5xl font-bold text-primary mb-2">
                  {study.metricValue}
                </div>
                <CardTitle className="text-lg">{study.metric}</CardTitle>
                <CardDescription className="text-sm">
                  {study.client} &middot; {study.industry}

              </CardHeader>
              <CardContent>
                <p className="text-muted-foreground mb-4">
                  {study.description}
                </p>
                <Link
                  href={`/case-studies/${study.slug}`}
                  className="inline-flex items-center text-primary font-medium hover:underline"
                >
                  Read case study
                  <ArrowRight className="ml-1 h-4 w-4 transition-transform group-hover:translate-x-1" />

    </section>
  )
}
```

- Card hover effect (shadow-lg) with group hover for arrow animation
- Links to future case study detail pages (/case-studies/[slug])
- Responsive grid: 1 col mobile, 2 col tablet, 3 col desktop

```typescript
      <div className="container px-4 md:px-6">

          <p className="mt-4 text-lg text-muted-foreground max-w-2xl mx-auto">
            From discovery to deployment in weeks, not months
          </p>
        </FadeIn>

        <StaggerContainer className="grid gap-8 md:grid-cols-2 lg:grid-cols-4">
          {steps.map((step) => (
            <StaggerItem key={step.number} className="text-center">
              {/* Large step number as visual anchor */}
              <div className="text-5xl md:text-6xl font-bold text-primary/20 mb-4">
                {step.number}
              </div>
              <h3 className="text-xl font-semibold mb-2">{step.title}</h3>
              <p className="text-muted-foreground">{step.description}</p>
            </StaggerItem>
          ))}
        </StaggerContainer>
      </div>
    </section>
  )
}
```

Key design decisions:
- Background muted/30 for visual section separation from surrounding white sections
- Large step numbers (5xl-6xl) with low opacity (primary/20) as visual anchors
- 4-column grid on desktop for 4 steps - natural fit
- Server Component with animation wrappers
- StaggerContainer for sequential reveal effect
- Concise descriptions - each step is one sentence
- "weeks, not months" subheadline addresses common concern about implementation timeline
