  <action>
Run create-next-app to initialize the project in the current directory:

```bash
npx create-next-app@latest . --typescript --tailwind --eslint --app --src-dir --import-alias "@/*" --use-npm
```

If directory is not empty, the command will ask to proceed - answer yes to overwrite.

This creates:
- Next.js 15 with App Router (default)
- TypeScript configuration
- Tailwind CSS v4 with CSS-based configuration
- ESLint for linting
- src/ directory structure
- Import alias @/* pointing to src/

Do NOT use --turbo flag (Turbopack is opt-in at dev time, not install time).
Do NOT manually create files - let create-next-app scaffold everything.
  </action>

Run `npm run dev` and verify:
1. Server starts on localhost:3000
2. Browser shows Next.js starter page
3. No TypeScript or ESLint errors in terminal

  </verify>
