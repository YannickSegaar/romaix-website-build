  <action>
Update globals.css to use brand color #587C74 converted to OKLCH format.

The brand color #587C74 converts approximately to: oklch(0.52 0.05 170)

In src/app/globals.css, find the :root section and update the --primary variable:

```css
:root {
  /* ... other variables ... */

  /* Brand color #587C74 in OKLCH */
  --primary: oklch(0.52 0.05 170);
  --primary-foreground: oklch(0.985 0 0);

  /* ... rest of variables ... */
}
```

Also update the .dark section if present:

```css
.dark {
  /* ... other variables ... */

  /* Brand color - slightly lighter for dark mode */
  --primary: oklch(0.55 0.05 170);
  --primary-foreground: oklch(0.145 0 0);

  /* ... rest of variables ... */
}
```

CRITICAL: Use OKLCH format (not hex or HSL). shadcn/ui v2 requires OKLCH for proper color handling and opacity modifiers like bg-primary/50.

Do NOT change other CSS variables unless needed for brand consistency.
Do NOT remove Tailwind directives (@tailwind base, components, utilities).
